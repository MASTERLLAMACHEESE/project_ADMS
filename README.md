# project ADMS
Repository for ADMS (**A**nalog **D**igital **M**odular **S**ynth)

Features:
- Pad w/ USB support
- 16 beat sequencer (every beat supports up to 4 sounds)
- Controllable BPM
- Sound pitch changing
- LED indication

Requirements: 
- pydub
- pynput - *for PC*
- Raspberry Pi 4 w/ Raspberry Pi OS Lite
